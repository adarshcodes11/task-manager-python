#  Task Manager (Python)

##  Overview
This project is a simple Task Manager application built using Python. It helps users manage daily tasks efficiently through a menu-driven interface.

The project demonstrates core programming concepts such as data structures, conditional logic, and problem-solving.

---

##  Features
-  Add new tasks  
-  View all tasks  
-  Remove tasks  
-  Mark tasks as completed  
-  Set task priority (High / Medium / Low)  

---

##  Technologies Used
- Python  
- Lists (Data Structures)  
- Conditional Statements  
- Functions
